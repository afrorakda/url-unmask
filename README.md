# URL Unmask
**Ultimate URL Decoder & Explorer**

A high-performance utility for developers to instantly decode, encode, and analyze complex URLs. Built for speed, deep inspection, and zero-clutter debugging.

## 1 Tool / 1 Action
- **Instant Analysis**: Paste any URL or encoded string to see both Unmasked (Decoded) and Masked (Encoded) results simultaneously.
- **Smart Deep Unmask**: Automatically detects and handles "Double Encoding" (e.g., `%25E3...`). If a second layer of encoding is found, the tool reveals the true content automatically.
- **Parameter Explorer**: Intelligently breaks down long query strings into a clean, readable list of Key-Value pairs.

## Features
- **Auto-Expanding Input**: The interface adapts to the length of your URL, ensuring the entire string is always visible.
- **1-Tap Individual Copy**: Every result—including individual parameters—can be copied with a single tap.
- **Privacy First**: Processing happens entirely in your browser. No data is sent to any server, making it safe for sensitive tokens and internal URLs.

## Design Philosophy
- **Minimalist UI**: Pure white background with #333 main accents and 15px rounded corners.
- **Mobile First**: Optimized for quick debugging on the go, whether on iPhone or Android.
- **Zero Latency**: No heavy libraries, no ads, no tracking. Just the logic you need.

## How to Install
1. Open the tool in your mobile browser.
2. **iPhone (Safari)**: Tap 'Share' -> 'Add to Home Screen'.
3. **Android (Chrome)**: Tap 'Menu (⋮)' -> 'Add to Home Screen'.
